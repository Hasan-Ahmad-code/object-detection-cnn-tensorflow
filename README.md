This repository contains an implementation of a simple object detection system built using TensorFlow and Keras. The model is designed to detect three object classes: car, dog, and person, while also predicting bounding box coordinates for object localization. The dataset was prepared and annotated using Roboflow and exported in Pascal VOC XML format. The project demonstrates the complete computer vision workflow including dataset preparation, annotation parsing, model training, and evaluation. A MobileNetV2-based architecture is used as the backbone for feature extraction. The network is trained to perform both object classification and bounding box regression simultaneously. This project was developed as part of my learning journey in Computer Vision and Deep Learning.
